# Break & Grow

Мотивирующий фокус-таймер: каждые 45 минут напоминает о глобальных целях и призывает перезарядиться — зарядкой, йогой или визуализацией цели. Любимые отрезки треков и видеофоны создают атмосферу.

## Быстрый старт

**Без установки:** открой `dist/index.html` двойным кликом — это полная сборка приложения в одном файле.

**Режим разработки:**

```bash
npm install
npm run dev      # http://localhost:5173
npm run build    # соберёт dist/index.html (один файл)
```

## Возможности MVP

- Таймер фокуса (по умолчанию 45 мин) с тремя режимами перерыва: **Цели** (напоминание «ради чего»), **Зарядка** (случайная разминка), **Йога** (позы + дыхание)
- Минуты «на лету»: пресеты 15/25/45/60/90, свои минуты, ±5 во время сессии
- Уведомления браузера + звуковой сигнал (WebAudio, без файлов) + полноэкранный мотивирующий оверлей
- Музыка: свои mp3 с отрезками A–B по кругу, YouTube с отрезком, Spotify — этап 2
- Фоны: анимированные градиенты, своё видео (хранится локально в IndexedDB), YouTube-видео
- Кастомизация: акцентный цвет, тёмная/светлая тема, затемнение фона, длительности, автостарты, громкость, язык RU/EN
- **6 игровых стилей** (Кастомизация → Игровой стиль) с прогрессом сессии как игровой механикой:
  - *Классика* — glassmorphism, кольцо прогресса вокруг циферблата, нойз-текстура
  - *Пиксель-квест* — HUD `SCORE · монеты · WORLD · TIME`, герой бежит по уровню к флагу ровно по прогрессу фокуса и прыгает на нём в конце, пиксельный прогресс-бар из золота
  - *Лабиринт* — HUD `1UP · HIGH SCORE · жизни`, жёлтый едок съедает дорожку точек = твой прогресс, за ним три призрака
  - *Блокопад* — HUD `SCORE · LEVEL · LINES`, стек блоков растёт по мере сессии
  - *Космо-рейд* — HUD `SCORE<1> · HI-SCORE · корабли`, флот из 18 пришельцев редеет по мере фокуса (взрыв на каждом), CRT-сканлайны
  - *Неон-драйв* — дашборд `SPD/DIST/STAGE` (скорость растёт с прогрессом), солнце поднимается, сетка ускоряется
  - Метафоры очков: фокус-минуты → SCORE, сессии → монеты/уровни/жизни; hi-score сохраняется
  - `READY? → GO!` при старте и чиптюн-звуки (квадратная волна) в аркадных стилях, сатурированная пила в неоне. Все спрайты свои — оммажи, не копии
- Всё сохраняется локально (localStorage + IndexedDB), работает офлайн (кроме YouTube)
- Таймер переживает перезагрузку страницы; экран не гаснет во время фокуса (Wake Lock)

## Структура

```
index.html            — точка входа
vite.config.js        — сборка в один файл (vite-plugin-singlefile)
src/
  main.jsx            — монтирование React
  App.jsx             — раскладка: топбар, таймер, панели, дровер, оверлей
  store.jsx           — состояние + таймер-движок + персистентность
  i18n.js             — словари RU/EN (90 ключей)
  content.js          — упражнения, йога, аффирмации
  db.js               — IndexedDB для медиа-блобов
  audio.js            — сигналы WebAudio
  yt.js               — парсинг ссылок + IFrame API
  styles.css          — дизайн-система (темы, акценты, glassmorphism)
  components/
    TimerCard.jsx     — циферблат, режимы, минуты на лету
    GoalsPanel.jsx    — глобальные цели и «зачем»
    MusicPanel.jsx    — треки, отрезки A–B, YouTube-плеер
    BackgroundLayer.jsx — градиенты / видео / YouTube-фон
    BreakOverlay.jsx  — экран перерыва по режиму
    SettingsPanel.jsx — секция кастомизации
```

## Заметки

- Разреши уведомления при первом старте таймера — иначе будет только звук и оверлей.
- Видеофоны: горизонтальные ролики 1080p смотрятся лучше всего.
- Дальнейшее развитие — см. `ARCHITECTURE.md`.
